## About Goravel-Admin

使用百度amis 低代码平台, 来快速创建后台

无须你编写任何前端代码, 就能创建一个好看的后台页面

## Getting started

运行项目, 打开 localhost:8080/admin
创建数据库, 导入SQL文件

```
// Generate APP_KEY
go run . artisan key:generate
// 运行项目
go run .
```

登录
账号:admin
密码:admin
